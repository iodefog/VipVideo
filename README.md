## 软件功能介绍：

* 1.展示窗口
* 2.新建窗口
* 3.GoBack
* 4.GoFoward
* 5.切换下一个（支持快捷键）
* 6.切换上一个（支持快捷键）
* 7.复制链接（用于分享或者浏览器播放）
* 8.切换vip接口
* 9.退出
